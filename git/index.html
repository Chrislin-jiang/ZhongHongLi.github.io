# git的三个作用域

pwd 现有目录的当前路径

$ git config --local local只对某个仓库有效 $ git config --global global对当前用户的所有仓库有效 $ git config --system system对系统所用登录有效

显示配置config的配置，加 --list $ git config --list --local $ git config --list --global $ git config --list --system

$ git config --global --list 查看用户名跟邮箱
